# Online/Offline Budget Tracker

This is a budget tracker app that is used to handle your budget online or offline. The app is a great way to trackm deposits and expenses, and is able to track on and offline.


# Tech Used
- Npm Depenencies
    - Compression
    - Express
    - Mongoose
    - Morgan

# Installation
To install fork this repo from my github repo. When forked open your Editor and in the terminal type npm i, and then npm start to run the server.

The app has also been deployed to Heroku.
